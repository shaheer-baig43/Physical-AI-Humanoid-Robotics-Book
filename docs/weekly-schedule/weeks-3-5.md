# Weeks 3-5: ROS 2 Fundamentals

These three weeks will be a deep dive into the Robot Operating System (ROS 2).

- **ROS 2 architecture and core concepts:** Understand the fundamental building blocks of ROS 2, including the underlying DDS communication system.
- **Nodes, topics, services, and actions:** Learn about the different communication patterns used in ROS 2 and when to use them.
- **Building ROS 2 packages with Python:** Get hands-on experience creating your own ROS 2 packages and Python nodes.
- **Launch files and parameter management:** Discover how to manage complex robotic systems with multiple nodes and configurations using launch files.
