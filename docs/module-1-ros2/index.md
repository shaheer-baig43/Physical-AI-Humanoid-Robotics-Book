# Module 1: The Robotic Nervous System (ROS 2)

**Focus:** Middleware for robot control.

### Module Objectives
*   Understand the fundamental architecture of ROS 2.
*   Learn to create and manage ROS 2 nodes, topics, and services.
*   Develop Python-based ROS 2 controllers using `rclpy`.
*   Master the Unified Robot Description Format (URDF) for describing humanoid robots.
