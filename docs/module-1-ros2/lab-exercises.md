# Lab Exercises for Module 1

This section will contain lab exercises for Module 1: The Robotic Nervous System (ROS 2).

*   **Exercise 1:** Installing ROS 2 and setting up a workspace.
*   **Exercise 2:** Creating a simple ROS 2 package.
*   **Exercise 3:** Writing a Python node that publishes a "Hello World" message to a topic.
*   **Exercise 4:** Writing a Python node that subscribes to the "Hello World" topic and prints the message.
*   **Exercise 5:** Creating a simple service that adds two integers.
*   **Exercise 6:** Writing a simple action server and client.
*   **Exercise 7:** Building a basic URDF for a two-link arm and visualizing it in RViz.
