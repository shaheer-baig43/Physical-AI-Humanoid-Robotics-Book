# Lab Exercises for Module 2

This section will contain lab exercises for Module 2: The Digital Twin.

*   **Exercise 1:** Setting up a Gazebo world with basic shapes and physics.
*   **Exercise 2:** Importing a URDF model into Gazebo and applying forces.
*   **Exercise 3:** Adding a simulated LiDAR sensor to a robot in Gazebo.
*   **Exercise 4:** Visualizing sensor data from Gazebo in RViz.
*   **Exercise 5 (Optional):** Creating a simple scene in Unity and importing a robot model.
