# Isaac ROS

**Isaac ROS** is a collection of hardware-accelerated packages for the Robot Operating System (ROS) 2. It leverages NVIDIA's GPU technology to significantly speed up common robotics perception tasks.

## Key Features

- **Hardware-Accelerated Perception:** Provides ROS 2 packages that are optimized to run on NVIDIA GPUs, leading to massive performance gains.
- **VSLAM (Visual SLAM):** Includes packages for Visual Simultaneous Localization and Mapping, allowing robots to navigate in unknown environments using camera data.
- **Navigation:** Integrates with Nav2 for robust and efficient path planning and navigation.
- **Object Detection and Recognition:** Offers pre-trained models and tools for detecting and identifying objects in the robot's environment.
